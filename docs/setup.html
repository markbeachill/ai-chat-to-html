Getting set up

Setting up Python.

The tool is a small Python program, so you need Python on your computer to run it. This is a one-time setup, and it's free.

What is Python?

Python is a widely used programming language. Many tools — including this one — are written in it. Installing Python gives your computer the ability to run those programs; you don't need to learn the language to use them.

Do I already have it?

Many computers, especially Macs and Linux machines, already have Python. To check, open a terminal (see below) and type:

python3 --version

If you see something like Python 3.11.4, you're ready — skip to the instructions. If you get an error or "command not found", install it using the steps below.

Installing it

The simplest way is the official installer.

  1. Download

    Go to python.org/downloads and click the button to download the latest version for your system. It detects Windows or Mac automatically.

  2. Run the installer

    Open the downloaded file and follow the prompts. On Windows, tick "Add Python to PATH" on the first screen — this lets you run it from the terminal. On a Mac, the defaults are fine.

  3. Check it worked

    Open a terminal and run python3 --version again. Seeing a version number means you're done.

Opening a terminal

The terminal is where you type the command to run the tool.

  • Windows: press Start, type PowerShell, and open it.
  • Mac: press Cmd+Space, type Terminal, and open it.
  • Linux: open your Terminal app from the applications menu.

One more thing: the dependency

The tool uses one small Python add-on. Once Python is installed, run this from the project folder to fetch it:

pip install -r requirements.txt

That's the last setup step. From here, the instructions take over.

Stuck? The official Python setup guide has detailed help for every system, including screenshots.
