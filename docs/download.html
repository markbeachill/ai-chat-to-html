Before it will run

The tool needs Python 3 and one small add-on. If you're new to Python, the setup page walks through installing it and opening a terminal. Once Python is ready, unzip the download and, in that folder, run:

pip install -r requirements.txt
python3 aichatprocess.py

Then follow the instructions to make your first page.
