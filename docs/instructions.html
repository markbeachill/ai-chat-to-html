Instructions

How to use it

From a copied conversation to a finished web page in three steps: copy & paste, edit, convert. The first two are just text editing; the third is one command.

The simple version

# put your conversation in aichat.txt, then:
python3 aichatprocess.py
# open aichat.html

Everything beyond this — themes, a Word version, maths — is an optional extra, covered at the end.

Before you start

You need Python 3 and the tool's one dependency. New to Python? The setup page walks through installing it. Then, from the project folder:

pip install -r requirements.txt

That installs Python-Markdown, which the tool uses to convert your text. Nothing else is required.

The three steps

  1. Copy & paste from the chat

    Copy the conversation turn by turn into a plain text file (the default name is aichat.txt). Mark each turn with a line that says who is speaking:

    # **[USER]**
Tell me about gravity?
# **[CHATBOT]**
Gravity is the force that attracts objects with mass...

    The marker only needs to contain [USER] or [CHATBOT] — the exact punctuation doesn't matter. Everyday Markdown formatting is kept (see the quick Markdown guide). Images and attachments are ignored.

    Prefer to edit in a document tool? Save the file as aichat.md instead of aichat.txt and open it in a Markdown-aware editor such as LibreOffice Writer. With no options, the tool uses aichat.txt if it's there and otherwise falls back to aichat.md, so either name works.

  2. Edit

    The text file is where you do the real editing: fix wording, shorten long answers, remove anything you don't want. The speaker lines keep the conversation easy to follow while you work. The Markdown guide covers the few symbols you might need.

  3. Convert

    Run the tool. With no options it reads aichat.txt and writes aichat.html plus aichat.css:

    python3 aichatprocess.py

    Open aichat.html in any browser and you'll see the conversation laid out as a chat. Optional extras: add --theme dark (or minimal, warm) to change the look, --word-output to also get a version for Microsoft Word, and --math to typeset any formulae written between $…$ or $$…$$ (web page only).

After you convert

That's the core done. Two optional follow-ups, if you need them:

Polish the page. Open the generated page in an HTML editor or preview tool to make final visual tweaks with the styling already applied — the content editing is already done.

Reuse it elsewhere. Inside the page, the chat section sits between two clearly labelled comment blocks (COPY CHAT HTML START / END). Copy everything between them into another page, and either link aichat.css or paste its rules into your stylesheet. The styles are namespaced, so they won't clash with the rest of the page.

A note on maths. The tool doesn't render mathematical notation automatically - it will just be plain text as typed, However, if you enclose the formula with "$$$" either side and use the "--math" option then it will render as a formula.

That's it

Most of the work is in the first two steps — getting the conversation into clean text. The conversion itself is a single command you can re-run any time you change the transcript.
