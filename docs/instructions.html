AI Chat to HTML

Turn a marked AI chat transcript into a clean HTML page.

Home Instructions Collector Prompt Reference

Instructions

Start by creating a file called aichat.md. Copy the conversation into the file and put # **[USER]** before each human message and # **[CHATBOT]** before each chatbot response.

# **[USER]**

Can you explain this?

# **[CHATBOT]**

Yes. Here is the explanation.

Edit the Markdown transcript until the example chat reads the way you want. Then run the processor:

python3 aichatprocess.py

The processor creates aichat.html and, if needed, aichat.css. Open the HTML file to preview the chat page.

Alternative capture method

You can also use the Conversation Collector Prompt. Paste it at the end of a chatbot conversation and ask the chatbot to produce a draft aichat.md. Check the result before processing.
