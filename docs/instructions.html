Instructions

How to use it

From a copied conversation to a finished web page in five stages. The first two are just text editing; the rest is one command and some optional polish.

Before you start

You need Python 3 and the tool's one dependency. New to Python? The setup page walks through installing it. Then, from the project folder:

pip install -r requirements.txt

That installs Python-Markdown, which the tool uses to convert your text. Nothing else is required.

The five stages

  1. Capture the chat

    Copy the conversation turn by turn into a plain text file (the default name is aichat.txt). Mark each turn with a line that says who is speaking:

    # **[USER]**
Tell me about gravity?
# **[CHATBOT]**
Gravity is the force that attracts objects with mass...

    The marker only needs to contain [USER] or [CHATBOT] — the exact punctuation doesn't matter. Everyday Markdown (paragraphs, lists, links, headings, code blocks) is kept. Images and attachments are ignored.

    Prefer a Markdown editor? Save the file as aichat.md instead of aichat.txt and edit it in your favourite Markdown app. With no options, the tool uses aichat.txt if it's there and otherwise falls back to aichat.md — so either name just works.

  2. Edit the transcript

    This text file is where you do the real editing: fix wording, shorten long answers, remove anything you don't want. The speaker lines keep the conversation easy to follow while you work.

  3. Convert it

    Run the tool. With no options it reads aichat.txt and writes aichat.html plus aichat.css:

    python3 aichatprocess.py

    Open aichat.html in any browser and you'll see the conversation laid out as a chat — your turns on one side, the chatbot's on the other. To point at different files or restyle, see the reference.

    Two handy extras: add --theme dark (or minimal, warm) to change the look, and --word-output aichat-word.html to also get a version you can paste into Microsoft Word.

  4. Polish the page

    Open the generated page in an editor or preview tool to make final visual tweaks with the styling already applied. The heavy editing is already done in stage 2, so this is just presentation.

  5. Reuse it elsewhere

    Inside the page, the chat section sits between two clearly labelled comment blocks:

    <!-- COPY CHAT HTML START -->
   ... the chat ...
<!-- COPY CHAT HTML END -->

    Copy everything between them into another page. Bring the styling too: either link aichat.css from that page, or paste its rules into your existing stylesheet. The styles are namespaced so they won't clash with the rest of the page.

A note on maths. The tool doesn't render mathematical notation. A formula written as plain text will appear exactly as typed, not as a typeset equation.

That's it

Most of the work is in stages 1 and 2 — getting the conversation into clean text. The conversion itself is a single command you can re-run any time you change the transcript.
