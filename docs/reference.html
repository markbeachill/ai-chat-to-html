Reference

Default files

The default source file is aichat.md. The default output file is aichat.html. The default CSS file is aichat.css. The default CSS template is aichat-template.css.

Command options

--source
--output
--css
--css-template
--write-css
--title

--write-css can be never, if-missing, or overwrite. If the external CSS template exists, it is copied when CSS is written. If the template is missing, an internal default CSS backup is used.

Image handling

External Markdown image links are preserved. Local, pasted, uploaded, internal, or non-portable image references are replaced with a placeholder.

Copy markers

The generated HTML page is a full page, not a snippet. The reusable chat section is surrounded by large repeated HTML comments labelled COPY CHAT HTML START and COPY CHAT HTML END.
