Two kinds of output

From the same transcript, the tool can produce two different things, depending on where the conversation needs to go:

A chat web page

A self-contained HTML page styled like a chat window — to publish, embed, or copy into another site. Comes in four themes.

A Word version

A clean transcript built to paste straight into Microsoft Word, in your choice of default, compact or plain. See the example.

Why use it

Write in plain text

Edit the conversation as Markdown — tidy wording, trim answers, rearrange — before any HTML exists.

Get a finished page

One command produces a complete HTML page and a matching stylesheet. No setup beyond Python.

Reuse the snippet

The chat section is marked with clear start/end comments, so you can lift it into any other page.

Restyle everything

Swap in a stylesheet template to change the look of every page the tool generates. See the themes →
