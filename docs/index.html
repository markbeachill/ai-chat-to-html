A small tool with a simple job

Turn a chat into a web page — or a Word document.

Paste a conversation into a text file, run one command, and get it back two ways: a clean web page that looks like a chat window, and a transcript ready to paste into Microsoft Word.

How to use it Collector prompt See a full example Download

The simple version

# put your conversation in aichat.txt, then:
python3 aichatprocess.py
# open aichat.html

That's the whole core. Themes, a Word version and maths are all optional extras — add them only if you want them.

Why use it

Write in plain text

Edit the conversation as Markdown — tidy wording, trim answers, rearrange — before any HTML exists.

Get a finished page

One command produces a complete HTML page and a matching stylesheet. No setup beyond Python.

Reuse the snippet

The chat section is marked with clear start/end comments, so you can lift it into any other page.

Restyle everything

Swap in a stylesheet template to change the look of every page the tool generates. See the themes →

Get started

The whole process is three short steps: capture, edit, convert. Capture can be manual copy & paste, or you can use the optional Conversation Collector Prompt. From there to reusing the finished page.
