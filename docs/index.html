AI Chat to HTML

Turn a marked AI chat transcript into a clean HTML page.

Home Instructions Collector Prompt Reference

Simple purpose

AI Chat to HTML converts a manually prepared chat transcript into a complete HTML page and a separate CSS file.

The source file uses two clear markers: # **[USER]** and # **[CHATBOT]**. The processor turns those marked turns into a generic chat-window layout.

Basic workflow

Create or collect a transcript as aichat.md, edit it, run the Python processor, then open aichat.html. The reusable chat section is marked in the HTML source with large comment blocks so it is easy to copy into another page.

You can either copy and paste turns manually, or use the Conversation Collector Prompt to ask a chatbot to produce a draft transcript in the correct format.
