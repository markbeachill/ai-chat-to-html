Two ways to convert

Turn AI chats into HTML pages.

Paste a marked conversation and convert it online, or use the Python tool for a repeatable local workflow. Both routes use the same transcript markers and can produce chat-style or Word-friendly HTML.

Fastest route

Do it online

Paste your marked transcript into the browser-only converter, choose chat-style or Word-friendly HTML, preview the result, and copy the generated HTML.

No install. No command line. Nothing uploaded.

Open online converter

Local route

Use the Python tool

Download the tool and run it locally when you want file-based processing, repeatable outputs, CSS templates, and a more controlled workflow.

Best for regular use or repository workflows.

The shared input format

Both routes start with the same simple Markdown-style transcript. Use # **[USER]** before user turns and # **[CHATBOT]** before chatbot turns.

# **[USER]**

Can you explain this?

# **[CHATBOT]**

Yes. Here is the explanation.
